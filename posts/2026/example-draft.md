이 파일은 `cha-amu/storage` 저장소에서 직접 포스트를 작성하는 예제입니다.

`status: draft`라서 공개 사이트에는 표시되지 않습니다. 공개하려면 `published`로 바꿉니다.

이미지는 storage repo 안의 상대경로로 연결합니다.

```md
![이미지 설명](../../assets/images/2026/sample.png)
```

파일도 같은 방식으로 연결합니다.

```md
[PDF 파일](../../assets/files/2026/sample.pdf)
```

외부 URL도 그대로 사용할 수 있습니다.

```md
[외부 링크](https://example.com/)
```
