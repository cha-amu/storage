# cha-amu storage

Static source and mirror repository for `https://cha-amu.github.io/`.

## Layout

```txt
posts/
  2026/
    example-draft.md

assets/
  images/
    2026/
  files/
    2026/

manifests/
  posts.json
  assets.json

manifest.json
```

- Write posts as Markdown under `posts/YYYY/`.
- Put image files under `assets/images/YYYY/`.
- Put PDFs, ZIPs, text files, and other downloadable files under `assets/files/YYYY/`.
- Manifest files are generated by this repo's `Sync storage repo` workflow and should not be edited by hand unless bootstrapping the repo.

The sync workflow runs in this storage repo. It reads Google Sheets through Apps Script, writes missing posts/assets into this repository, regenerates manifests, and commits the result with this repo's built-in `GITHUB_TOKEN`.

See the site repo document `docs/storage-authoring.md` for the full authoring format.
